basic commands
ls : List file
ls -Force : List hidden files
pwd : Shows where you are in which directory
cd Folder_name : Move into the Folder_name
cd .. : Move to parent Folder
New-Item -ItemType Directory Folder_name : Makes new folder
New-Item FileName.txt : Creates an empty file
Remove-Item FileName.txt : Deletes a file
Remove-Item FolderName -Recurse -Force : Deletes folder and content
Move-Item Source Destination : Move files
